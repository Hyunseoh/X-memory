# 서비스 소개 😄
- X-memory : 과거 사진의 의미를 다시 상기시킬 수 있도록 도와주는 앱 이름 선정
- 객체 인식으로 자동으로 태그를 생성하고, 생성된 태그로 분류 및 검색 가능
- 태그를 통한 사용자 일상 데이터 분석과 같은 새로운 가치 창출
- AI 자동 태그 외 메타데이터, 사용자 인풋 태그를 받아 보다 자세한 검색 및 분석 서비스 제공

# 서비스 목적 🔭
- 사용자의 잊혀진 사진들에 대한 추억 상기 효과, 사진 관리 용이성 제공, 태그 분석을 통한 새로운 가치 창출

# 데이터 분류 방법 📝
- 사진 메타 데이터 태그
- AI 이미지 분석을 통해 검출한 객체 태그
- 사용자가 직접 입력한 태그
